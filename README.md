# InstantWeather Android App

InstantWeather is an Android application that provides accurate weather forecasts with a simple and clean UI.

## Prerequisites
Make sure you have the following installed on your system:

- [Android Studio](https://developer.android.com/studio) (Narwhal 4 Feature Drop or newer recommended)
- [Java JDK 11+](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) or OpenJDK
- [Gradle](https://gradle.org/) (optional, Android Studio comes with Gradle)
- Internet connection for downloading dependencies

---

## Building the App
1. **Clone the repository**
2. **Open the project in Android Studio**
3. **Let Android Studio sync Gradle and download dependencies**
4. **When all dependencies are downloaded, click the run button on Android studio to run the app**

---

## Dependencies
The app uses the following key technologies:

- Room – for local database persistence
- Hilt – for dependency injection
- Retrofit – for network requests
- Moshi – for JSON parsing
- Kotlin Coroutines – for asynchronous operations
- Timber – for logging
